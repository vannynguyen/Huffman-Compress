1. For the .txt file you want to compress, created two duplicates.

2. Add “-compressed” to the end of the first duplicate’s name.

3. Add “-decompressed” to the end of the second duplicate’s name.

4. Run “HuffmanCompressor.java” and select the original file you want to compress.

5. Verify the compression and decompressions by checking the sizes of the compressed and decompressed files.
